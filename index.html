comands

    git commands

  • git clone pat-to-ropository - копирует проект с GitHub на комп
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавляет файл в локальный Git
  • git commit -am "added new file index.html" - зафиксировать все изменения и дать им комментарии
  • git push - отправляем все изменения с коммитами на GitHub
  • git pull - забрать все изменения с GitHub к себе в проект
  • git add . - добавляет в гит все что есть в папке
  • gitk --all& - графическая интерпретация консоли
  • ctrl + insert - копируем в буфер
  • shift + insert - вставить из буфера
  • git checkout commit-number - переключиться на нужный коммит commit-number написан в строчке SHA1 ID
  • git checkout file-path - отменяет изменения в файле с прописанным путем
  • git log - просмотр коммитов в этом дереве
  • q - выход из git log
  • git reset - удаляет коммит
